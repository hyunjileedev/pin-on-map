## 🛠 사용 기술

- HTML+CSS
- JavaScript
- Window APIs: 좌표, 사이즈 관련

## 🎯 주요 기능

- 마우스 이동 시 커서를 따라 핀 이동
- 마우스 클릭 시 클릭된 위치에 핀 고정

## 📝 새롭게 배운 점

- 이미지 요소의 DOMRect를 받을 때 load 전에 받는 경우 정확한 값을 받을 수 없음 ⇒ 항상 정확한 값을 받으려면 load 이벤트 이후에 받아야 함
